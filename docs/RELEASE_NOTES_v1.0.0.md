## PiChat v1.0.0

First public release of PiChat — a native macOS client for the pi coding agent.

### Highlights

- Native SwiftUI desktop app for `pi --mode rpc`
- Tool streaming, queue view, and session stats
- Model/thinking controls and project switching
- Extension UI dialogs support
- Config editors for settings/models/auth
- DMG packaging for easy install

### Distribution

Download: **PiChat-macOS.dmg** from this release.

### Requirements

- macOS 14+
- `@mariozechner/pi-coding-agent` installed globally
