# PiChat Windows v1.0.5

Runtime reconnect hotfix.

## Fixed

- Fixed prompt failures when the `pi` RPC process had stopped before sending a message.
- Windows now reconnects `pi` automatically from the prompt path before submitting the message.

## Install

Download and run:

- `PiChat-Windows-Setup-1.0.5-x64.exe`

## Security note

This Windows release is not code-signed yet, so Windows SmartScreen may warn on first launch. Click **More info → Run anyway** only if you downloaded the installer from the official GitHub release.
