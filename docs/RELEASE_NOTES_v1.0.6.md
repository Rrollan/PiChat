## PiChat v1.0.6

Browser bridge setup release.

### Added

- Browser Assistant settings section for Browspi integration
- Native Messaging host installer for `com.browspi.pi_bridge`
- Bundled Browspi native host scripts for local Pi model and chat access
- Digit-only Browser UID support for easier Browspi pairing

### Fixed

- Open Chrome Extensions now opens `chrome://extensions/` through the default browser
- Improved clickable areas and hover feedback in model, thinking, tool, and side-panel controls

### Installation

Download `PiChat-macOS.dmg`, drag `PiChat.app` into `Applications`, and choose **Replace** if prompted.

To connect Browspi: open **Settings → Browser**, paste the Browspi Browser UID, install the native bridge, reload Browspi, then press **Connect** in the Browspi side panel.
