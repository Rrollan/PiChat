## PiChat v1.0.1

Maintenance release focused on install/update UX.

### What's improved

- Added in-app **Update** button (checks latest GitHub release and opens DMG)
- Added warning banner when app is launched outside `/Applications`
- Improved dark theme user-message contrast
- Added custom app icon to `.app` bundle and DMG flow
- Improved DMG installer layout

### Installation

Download `PiChat-macOS.dmg`, drag `PiChat.app` into `Applications`, and choose **Replace** if prompted.
